Оцени по стоимости и длительности данной работы, применив среднюю оценку по рынку ИТ услуг.
1. В таблице "Таблица объектов" добавить новую кнопку "Авто масс. добавление"
2. Нажав на кнопку "Авто масс. добавление" появляется всплывающее окно "Массовое добавление объектов"
окно "Массовое добавление объектов" состоит из пяти выпадающих меню, четырех кнопок, двух фильтров.
3. Создать таблицу - "Таблица объектов" из двеннадцати полей.
4. Во всплывающем окне "Массовое добавление объектов" пользователь заполняет параметры и нажимает на
кнопку "Создать", после чего, парсер собирает данные с площадки и добавляет в новые строчки
в таблице "Таблица объектов".
5. во время работы запроса должна работать анимация, показывающая работу парсера.
6. В парсере необходимо реализовать функцию проверки на дубликаты,
во избежание забивание строк в базу данных одинаковыми объектами.
7. Проверка при сохранении объектов после работы парсера в базу,
должна происходить по значениям  в "Email" и "Общий телефон"
8. По окончанию работы парсера есть окно - уведомление о результате из пяти параметров.
9. Парсинг должен работать на платформе "Яндекс карты".
